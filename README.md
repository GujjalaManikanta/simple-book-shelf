# Simple Book Shelf

A clean and simple library management system to add, search, sort, and manage your book collection.

## Project info

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To work locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   ```

2. **Navigate to the project directory**
   ```bash
   cd simple-book-shelf
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

## Deployment

Pushed changes can be deployed by running the `deploy.ps1` script to deploy to GitHub Pages.
